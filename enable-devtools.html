Pemulihan Akses DevTools

Gunakan halaman ini untuk mengaktifkan kembali DevTools yang telah dinonaktifkan pada website yang Anda kunjungi. Pilih salah satu metode di bawah ini:

Metode 1: Bookmark Instant

Seret bookmark ini ke bilah bookmark browser Anda, kemudian klik saat Anda berada di website target:

Aktifkan DevTools

Cara Penggunaan: Seret link di atas ke bilah bookmark browser Anda, kemudian klik bookmark tersebut saat berada di website yang memblokir DevTools.

Metode 2: Kode Konsol

Salin kode berikut dan tempelkan ke konsol browser (jika masih bisa dibuka):

(function() {
  console.log('üîç Memulai pemulihan DevTools...');
  
  // Kembalikan console API
  const originalConsole = {
    log: console.log.bind(console),
    error: console.error.bind(console),
    warn: console.warn.bind(console),
    info: console.info.bind(console),
    debug: console.debug.bind(console),
    clear: console.clear.bind(console)
  };
  
  Object.keys(originalConsole).forEach(key => {
    console[key] = originalConsole[key];
  });
  
  // Ganti addEventListener
  const originalAddEventListener = window.EventTarget.prototype.addEventListener;
  window.EventTarget.prototype.addEventListener = function(type, listener, options) {
    if (type === 'keydown' || type === 'keyup' || type === 'keypress') {
      const originalListener = listener;
      const newListener = function(event) {
        if ((event.key === 'F12') || 
            ((event.key === 'I' || event.key === 'i' || event.key === 'J' || event.key === 'j' || event.key === 'C' || event.key === 'c') && 
             (event.ctrlKey || event.metaKey) && event.shiftKey)) {
          console.log('üîë DevTools shortcut diizinkan');
          return;
        }
        return originalListener.apply(this, arguments);
      };
      return originalAddEventListener.call(this, type, newListener, options);
    }
    return originalAddEventListener.call(this, type, listener, options);
  };
  
  // Nonaktifkan deteksi DevTools
  Object.defineProperty(window, 'devtools', {
    get: function() { return undefined; },
    set: function() { },
    configurable: true
  });
  
  // Bersihkan interval
  for (let i = 1; i < 10000; i++) {
    window.clearInterval(i);
  }
  
  console.log('‚úÖ Mekanisme pembatasan DevTools telah dihapus!');
})();

Metode 3: Jalankan Langsung

Klik tombol di bawah untuk menjalankan script pemulihan DevTools di halaman ini:

Script berhasil dijalankan! Coba buka DevTools dengan F12 atau Ctrl+Shift+I (Windows) / Cmd+Option+I (Mac)

Jika Masih Gagal:

  1. Coba buka situs target di mode Penyamaran/Incognito
  2. Gunakan browser lain (Firefox sering lebih permisif untuk DevTools)
  3. Pada Chrome, coba tambahkan parameter --disable-features=BlockInsecurePrivateNetworkRequests saat meluncurkan browser
  4. Coba akses melalui shortcut keyboard langsung: F12 atau Ctrl+Shift+I (Windows/Linux) / Cmd+Option+I (Mac)

Bagaimana Cara Kerjanya?

Script ini bekerja dengan cara:

  1. Mengembalikan fungsi asli console API yang mungkin telah diganti
  2. Mengganti fungsi addEventListener untuk mencegah pemblokiran shortcut keyboard DevTools
  3. Menghapus properti yang digunakan untuk mendeteksi DevTools
  4. Membersihkan interval yang mungkin digunakan untuk memeriksa DevTools
  5. Mencegah pendeteksian melalui perbedaan ukuran jendela

Tool ini dibuat untuk tujuan pembelajaran dan debugging saja.
